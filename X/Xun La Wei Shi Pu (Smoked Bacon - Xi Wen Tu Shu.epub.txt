# 广式腊肠

原 料:

瘦肉1 200克,肥肉300克,干肠衣25克,细麻绳数条,玫瑰露酒100毫升,上等生抽200毫升,糖、盐各50克,硝盐2克。

制作过程:

1.将瘦肉、肥肉分别洗净,抹干水分,切成指甲般大小的肉片。

2.将味料放于大碗中和匀,放入肥肉、瘦肉片,加入半杯水拌匀,腌半小时。

3.肠衣套在水龙头上,冲净内外;用一个灌肉器穿入肠衣口处,把腌肉徐徐放入且不宜太紧实。用一支铁针插入肠中数次;肠灌好后头尾打结,用麻绳每隔16.5厘米长的地方扎紧成双形。

4.将灌好的腊肠放入滚水中汆水,再取出放在太阳猛烈的地方晒数天即成(如果想腊肠有香味,则需放入罐里封藏数天)。

# 传统广式腊肉

原 料:

切成条状的五花肉5 000克;盐125克,糖200克,汾酒100毫升,生抽200毫升,猪油50毫升,硝盐15克。

制作过程:

1.将五花肉先用温水洗净,再用冷水冲过,晾干水分。

2.把盐、糖、酒、生抽、硝盐等拌匀,将五花肉腌4小时。

3.在每块五花肉条头开一小孔,用细绳穿上,然后用猪油给五花肉上色(注意上色不能过久,过久会使腊肉色暗)。

4.将五花肉经细绳穿于竹竿,白天晒于日光下,晚上把腊肉放入火柜(火柜温度约为60。C~70。C)。

5.经过日晒与入火柜3~4次,瘦肉部分发硬便成腊肉。

# 腊鸭

原 料:

鸭1只(约1 800克),盐250克,乙基麦芽酚4克。

制作过程:

1.将鸭剁去鸭翼,自膝下斩去鸭脚。

2.从鸭尾股处下刀直穿至颈部丫叉骨处,将胸膛破开。

3.从颈项下刀将两边膀骨割断,并把两边肋骨各割断6~7条。

4.从鸭嘴处下刀把鸭下巴、鸭舌切去。

5.将盐调匀,在鸭的头、颈、身等处搓匀,再用50克盐分置于翼膊骨处和鸭腿骨底面,腌一个晚上。

6.第二天把腌过的鸭放在冷水中浸泡2~3个小时,以鸭咸味变淡为度。起盆上棚,置烈日下曝晒。

7.待鸭身爽透,再穿上绳圈挂晒于竹竿上。

# 腊味杏鲍菇

原 料:

腊肉250克,杏鲍菇200克,葱、蒜、青红椒、盐、蚝油各适量。

制作过程:

1.杏鲍菇洗干净后,用手撕成长条;腊肉切片;青红椒切好备用。

2.炒锅无油烧热,倒入撕好的杏鲍菇,干炒,炒软,盛出备用。

3.炒锅洗净上火,放油,炒香腊肉,加入葱、蒜,略加水,焖2分钟。

4.随后加入切好的青红椒炒出香味,倒入炒过的杏鲍菇,翻炒均匀,最后加入耗油、盐调味即可。

小贴士

杏鲍菇,又名刺芹侧耳,菌肉肥厚,质地脆嫩,是近年来开发栽培成功的集食用、药用、食疗于一体的珍稀食用菌新品种,被称为"平菇王",具有杏仁香味和如鲍鱼的口感,适合保鲜、加工,深得人们的喜爱。

# 腊鲮鱼

原 料:

鲮鱼6条(每条约750克),细麻绳6条,纱纸6张。味料1:磨豉酱500毫升,生姜蓉100克,糖100克,味精20克,大青蒜2条(制成末),头曲酒15毫升;味料2:盐3汤匙。

制作过程:

1.把鱼剖开,除去鳞、鳃及肠脏,并在两侧各割一刀,用3汤匙盐腌匀,再用重的厚砧板石压一夜,备用。

2.把味料1放于大碗中搅匀为酱料,备用。

3.把腌过一夜后的鱼用清水冲洗净,抹干水分,用酱料把每条鲮鱼内外涂匀。

4.涂匀酱料后,用绳子分别穿上鱼头,挂在太阳下晒一天,待酱干时,则分别用纱纸把鱼封包,再把鱼放太阳中晒3~4天即成。

# 腊鱼

原 料:

鲅鱼1 000克,海盐适量。

制作过程:

1.鲅鱼半解冻时,平剖开取出内脏,彻底清洗干净,然后晾干水分,均匀的撒上一层盐。

2.腌制一夜,中途可以翻面一下使得咸味均匀。

3.取出夹在夹子上在晴好的状态下晒2~3天即可。阴天可适当延长时间,看到肉收紧变的稍硬即可。

小贴士

鲅鱼鱼肉本身比较紧,所以不需要晒太长时间。若是用淡水鱼之类的肉质比较嫩的鱼,可以延长晒制时间。晒好的鱼如果短时间内不吃,冷冻储存。

# 腊鸡腿

原 料:

鸡腿2 000克,花椒、海盐各适量。

制作过程:

1.鸡腿洗净,先吊挂2个小时控干水分。

2.盐和花椒放入锅中炒至盐变黄,然后放入料理机稍微打碎一下。

3.趁热均匀的抹在晾干水分的鸡腿上,腌制24小时。中间可以上下翻面一次。

4.吊挂晒5~7天。

小贴士

具体是晒干还是阴干,味道差别不大,只是晒干的时间短点,阴干则需要多几天。晒2天就开始滴油,最后至肉发红变得很紧时即可。做好的腊鸡腿可以做蒸饭,或者加个土豆一起炖。

# 豆豉蒸腊鱼

原 料:

腊鱼1条,豆豉辣酱、蒜、葱、姜、猪油、料酒各适量。

制作过程:

1.腊鱼切块,提前浸泡一会,洗净沥干;蒜切碎;锅里热猪油,葱姜爆香加入豆豉辣酱,小火翻炒一下。

2.加入腊鱼,调入少许料酒翻炒均匀即可关火,加入蒜碎拌匀,倒入盘中。

3.上锅蒸40分钟即可。

小贴士

1.可以加腊肉一起蒸,味道更好。

2.可根据腊鱼的风干程度和咸度调节浸泡的时间。

# 芜湖腊味鸭肫

原 料:

鸭肫500克,盐、酱油、白酒、糖各适量。

制作过程:

1.选用整齐肥大鲜鸭肫,沿进食孔中间剖开,除去内容物,刮去黄皮和肫外附着的油皮,用少量食盐进行抹擦,搓揉,清水漂洗,直至无污物,无异味,沥水。

2.用部分盐将鸭肫逐个擦抹,放入容器中,腌制1天,取出沥去卤水,再放入另一容器中,加入盐、酱油、酒、糖、硝酸钠各2.5克,腌制2天,期间翻几次。

3.腌好的鸭肫,再用清水漂洗,去净杂质和污物,沥干水分,每10个穿成1串,晒至七成干,取下,整型。

4.将鸭肫平放在案上,用掌根用力压搓2~3次,使鸭肫两片的肌肉压平,再放阴凉通风处保存,一般小雪后制品可晾挂半年,过长会干缩变味。

# 腊猪腰

原 料:

猪腰子1 500克,酱油200毫升,白酒10毫升,糖4克,盐15克,五香粉5克。

制作过程:

1.将猪腰附着白色的油膏撕去,洗净后放入开水中片刻,取出去水用盆盛载,撒入盐1汤匙,拌匀腌约15分钟。

2.将老抽、糖、五香粉及酒全部放入足够容纳10条猪腰的盛器中,搅拌调和,留用。

3.把腌透的猪腰逐条取出,沥去过多的水,然后放入味料盛器中,并将其遍沾味料,经过一夜的浸腌,另用细绳穿起,挂到竹竿上晾晒,约经三四日干透即成。

小贴士

别名猪肾,味咸、平、无毒,归肾经;有理肾气,通膀胱,消积滞,止消渴之功效;主治肾虚所致的腰酸痛,肾虚遗精,耳聋,水肿,小便不利。一般人群均可食用,但血脂偏高者、高胆固醇者忌食。

# 腊鸭肝

原 料:

鸭肝500克,盐8克,糖40克,生抽10毫升,猪油(炼制)5毫升,白酒15毫升,姜汁10毫升。

制作过程:

1.鸭肝洗净,将盐、糖、生抽、猪油、白酒、姜汁放到一起,拌匀后腌鸭肝4小时。

2.然后把鸭肝铺在密竹筛上。

3.将竹筛连鸭肝上棚晒,每隔3小时翻转再晒,晚上入火柜焙至身爽便成。

4.食用时,蒸熟即可。

# 广州腊鸭翼

原 料:

鸭翅500克,盐50克。

制作过程:

1.将原只鸭翼洗净,用盐腌1天。

2.腌后取出,用清水浸泡半小时,以减轻其咸味。

3.然后,置于阳光下晒,或用火烘烤1天,第2天再晒。

4.3天后,即可制成腊鸭翼。

小贴士

食用时与鲜品制法相同,炖、煮、红烧,或煮汤均可。

# 腊鹅

原 料:

鹅1000克,盐150克,花椒10克,糖30克。

制作过程:

1.鹅(壮鹅)宰杀后,烫煺羽毛,从背上剖开,掏除内脏(另用)清洗干净,晒干水分。

2.盐炒去水分,均匀擦抹在鹅体上,肉厚处多擦,并用竹扦穿几个眼。擦抹后装入木盆中加盖,腌2天。

3.取出在太阳下挂晒1~2天,晒时用竹棍撑开,水分晒干后,即可收在避阳光,通风好的地方悬挂保存。

4.食用时切割部分采用适宜的方法烹制。

小贴士

1.有的地方加工时采用干拔羽毛的办法,注意不要把皮搞坏。

2.多数人对腊鹅的吃法是,皮和脂肪蒸食,瘦肉煎食,骨头煮食,煮时入芸豆或花生米味道更佳。

3.鹅宰杀时前禁食1~2天,可喂一些淡盐水。

# 腊关刀肉

原 料:

猪里脊肉500克,盐10克,糖50克,料酒10毫升,酱油15毫升。

制作过程:

1.把里脊肉去皮切成关刀型,用以上味料腌4小时。

2.每条肉条在一端穿一小孔,套上绳圈后上竹,晒及入火柜的操作与腊肠同,晒,焙四五天至瘦肉身硬便成。

3.入火柜忌近火位,近火位会变黄色不美观。

小贴士

硝水,就是硝酸钾或者硝酸钠的溶液可以使腌肉变红但是对人体有一定伤害,所以不能大量食用。

# 清蒸腊肉

原 料:

猪肋条肉(五花肉)1 000克,盐15克,花椒5克。

制作过程:

1.将猪肋条肉切成长条,用竹签扎些小眼,再用花椒和盐(都经过炒烫、晾至温热)进行揉搓,揉搓后放入瓷盆,皮朝下肉朝上,一层层码放,最上一层用重物压住。

2.每隔两天翻倒一次,腌10天后,改为每天翻倒一次,再腌4~5天,取出,

用绳穿上,吊挂通风处晾至半天。

3.在大铁锅内放入锯末,上面架铁痹子,把晾好的肉放置其上。

4.盖上锅盖,置于火上,至锯末受热冒烟时停火,将肉熏成黄色、干燥即可。

5.将制好的膜肉放入温水中泡软,刮去黄面,并刷去肉上的尖土。

6.用温水洗净,放入容器,入屉用旺火、沸水、足气蒸1~1.5个小时,取出,晾凉,切片装盘即可。

小贴士

需备锯末750克。

# 苦瓜腊肉

原 料:

苦瓜300克,腊肉150克,姜丝15克,蒜末10克,红辣椒10克,高汤30毫升,料酒10毫升,胡椒粉少许,生粉10克,盐与味精适量。

制作过程:

1.腊肉切片,用温水浸泡15分钟,苦瓜洗干净切片,红辣椒切段。

2.食油旺火起锅,放姜丝、蒜末、辣椒,炒出香味之后,再投入腊肉,翻炒。

3.烹入料酒,加入苦瓜片、30克高汤、胡椒粉、盐与味精,炒至只剩少许汤汁,勾芡即可。

小贴士

市场上卖的腊肉,咸味比较重,血压高的人要适当去咸,但不是说就不能吃。腊肉要炒出香味,才能加入苦瓜,苦瓜炒好之后,仍要带点脆性。

# 大蒜炒腊肉

原 料:

腊肉350克,大蒜苗100克,料酒、醋、酱油、味精、豆瓣酱各适量。

制作过程:

1.要提前把腊肉用水泡上,否则的话炒出来的菜会很咸,可以用开水,时间为20到30分钟;大蒜苗切段。

2.锅烧热,放油;然后放入腊肉翻炒,稍后再放入料酒和少量醋。

3.放入切好的大蒜苗继续翻炒,喜欢的话可以放入适量豆瓣酱,也可放入很少量水。

4.炒熟出锅前放入适量味精。

小贴士

同样方法可以做青椒炒腊肉,白菜炒腊肉等。

# 烟笋腊肉

原 料:

乡里腊肉300克,水发烟笋片120克;小葱20克,料酒4毫升,酱油3毫升,味精2克,干辣椒2克,豆豉3克,油70毫升。

制作过程:

1.将整条腊肉去皮,切成片;烟笋切片。

2.起锅放开水将腊肉焯一下,捞出。

3.另起锅放底油,投入豆豉、干辣椒、烟笋爆香。

4.放腊肉、料酒、酱油、味精、鸡汤烧开后用微火焖10分钟,下小葱焖2分钟,淋香油,撒上葱段即可。

# 尖椒腊肉

原 料:

尖椒150克、腊肉200克、盐、葱、姜、蒜各适量。

制作过程:

1.腊肉蒸15分钟,切成薄片。

2.炒锅放油烧热,放入蒜片红辣椒炒香。

3.放入腊肉炒2分钟。

4.放入辣椒炒2分钟。

5.放少许酱油和糖,翻炒装盘即可。

# 茶油蒸腊肠

原 料:

湘西腊肠300克,葱花、味精、味精、茶油、干红辣椒各适量。

# 蕨菜炒腊肉

原 料:

干蕨菜100克,腊肉250克,豆瓣酱、糖、盐、高汤、酱油、葱、味精各适量。

1.将蕨菜提前用温水泡发,清洗干净后,下入煮沸的水中焯烫5分钟左右捞出,过一遍清水,切成3厘米左右的段备用。

2.在焯烫蕨菜时,用小火煎一下腊肉,等腊肉出油,原本白色的油脂变得金黄,边缘卷起即可捞出。

1.腊肉本身就有盐味,豆瓣酱和酱油的添加要适量,如果觉得味道重酱油和盐可以省去。

2.如果能买到新鲜蕨菜,第一步可以省略。

# 萝卜干腊肉

原 料:

萝卜干150克,腊肉250克,干红辣椒、青蒜或蒜头、料酒、盐、酱油、味精各适量。

2.用蒜苗色彩更漂亮,用蒜头,菜没有绿色。味道是没有特别大的区别。

# 湘西腊肉

原 料:

冬至后宰杀的肉猪肉,将肉砍成肉条;盐、花椒、五香粉各适量。

制作过程:

1 .盐椒粉腌渍:肉、盐、花椒、五香粉按100∶5∶2∶1的比例,将盐椒粉拌匀,均匀抹在猪肉条表面,装入大缸腌渍,装满后用木板或薄膜盖住,不需翻动。腌渍时间为20~30天,让盐椒充分溶粘在肉条上。

2.熏烘:将腌渍好的肉条逐个穿绳悬挂在火堂上方的吊架上,以火堂中心点向四周扩展。肉条离火堂高1.2~1.6米,利用冬季农家在火堂上烧煮、取暖燃烧柴薪产生烟热气进行熏烘。熏烘的次数以自然炊食取暖次数为宜,日熏烘夜晾露,冷热相间。一般熏烘期为30~60天,燃料以炭薪林、用材林、锯木粉、枯饼、谷壳等为佳,切勿用垃圾或废纸屑、农膜烧熏。熏烘时火苗不宜过大过急,以防外干内生。肉条四周不宜用竹帘或农膜围栏。中途可将吊挂的肉条相互调换位置。

3.下架贮藏:经过一二个月的熏烘后,肉条逐渐失水变干,色泽由白变褐红色,此时可下架食用或贮藏。为了延长食用期,保证腊肉的正宗美味,可用四种方法贮藏:一是将熏烘好的肉条藏在谷堆或谷壳中;二是藏入锯木屑中;三是挂在通风干燥的壁板上;四是用稻草包裹放在干燥处。

4.加工时间一般在冬至后立春前。

3.加水淹没排骨,同时加少许茶叶、陈皮,先大火烧开,后中火煮约20分钟,收干水即可(注意此时的排骨是甜味较重,做到这一步即可食用,如果不想继续做下去,那么前面就少放点冰糖)。

4.蜂蜜加适量水稀释融化,将排骨放入其中浸泡后捞出。

5.锅垫上锡纸,加入5勺左右糖、10粒左右茶叶,架上蒸笼架,裹了蜜糖水的排骨放到蒸笼架上,盖锅盖,大火干烧待浓烟弥漫后,改中火5分钟即可。

小贴士

在熏之前的排骨甜味较重,经过这茶叶熏之后,变成了有点微苦,但吃完后又有丝丝甜味,回味悠长。

# 茶香熏兔

原 料:

肉兔1只(2.5~3千克),毕拨、良姜、桂皮、砂仁、花椒、肉豆蔻、大料、白芷、山楂、酱油、酱豆腐、面酱、盐、大蒜各适量。

制作过程:

1.肉兔按传统工序屠宰,放血,剥皮,开膛,除去内脏和四肢下部,用清水少洗净后,再用无毒线绳把两后肢绑成抱头状呈弓形固定。

2.选用毕拨、良姜、桂皮、砂仁、花椒、肉豆蔻、大料、白芷、山楂等各适量,装入纱布袋,煮兔时把调料袋放入锅内水中。

3.加入适量的酱油、酱豆腐、面酱食盐、大蒜等制成煮肉汤,这种调配的汤料具有口味好,防腐去腥,健胃,暖肠,调气,醒脑和增色等功效。

4.熟制:配好的佐料肉汤煮沸后放入兔,再加火煮沸,然后用慢火焖3~4小时,以兔肉熟烂而不破损为宜。

5.把煮好的兔捞出,置于特制的笼屉上,近代汽待熏。平时将煮肉汤盛于缸内保存,冷却去掉上层浮油,煮肉汤可连续使用。多年的肉汤叫"老汤",老汤质量的好坏是煮好兔的技术关键。

6.熏制:把铁锅清洗干净,在锅底部加入柏木或碎屑适量,糖少许,然后把待熏制的兔均屉上,再放入锅内,盖上锅盖,加火烧3~5分钟。待锅冒出缕缕青烟,闻到柏木香味时,揭开锅取出,即成。

# 麻香红茶熏鸡

原 料:

鸡腿500克,姜片10克,红茶15克,盐5克,小葱15克,红糖25克,八角5克,丁香5克,酱油少许,小米100克,料酒20毫升,香油15毫升,花椒5克。

制作过程:

1.鸡腿清洗干净后去骨,原料酒姜片少许酱油把鸡腿腌制半小时以上,用厨房纸将腌制好的鸡腿吸干水分。

2.将葱,盐与花椒均匀抹在鸡腿上。

3.将八角、丁香、小米、红茶与红糖一起放入温热锅内炒至香味出来,然后用锡纸或者烤箱纸包起来放在锅里,开大火。

4.将处理过的鸡腿放在蒸笼里盖上盖,放在熏料锅里,待浓烟弥漫后,改中火熏半小时即可出锅,出锅后刷少许香油。

# 烟熏猪舌头

原 料:

(A)猪舌头4条;(B)卤料:酱油,老抽,料酒,糖,葱,姜,八角,香叶,草果,花椒;(C)熏料:茶叶2大勺,糖2大勺,大米2大勺,橙皮2片,丁香2个,草果1个,姜1片。

制作过程:

1.先用盐把舌头抓洗几次,剪去肥油筋络等。

2.煮开一锅水,把猪舌头煮至变色,用清水冲洗后再用刀刮干净舌苔。

3 .锅中倒适量清水加入(B)中的卤料煮开,加入处理干净的猪舌头,煮至筷子刚能穿透,关火后把猪舌头留在卤汤里过夜,第二天再烧开,取出猪舌。

4.锅底垫一层锡纸,上面放入(C)中的熏料,盖上盖子先开大火,看见锅内冒烟后放入蒸架,摆上猪舌头。

5.盖上锅盖,看见锅内浓烟弥漫时,关小火熏2分钟后关火,不要打开盖子,让猪舌头慢慢吸收残余的烟,等熏烟都干净后取出猪舌,稍凉后切片即可食用。
